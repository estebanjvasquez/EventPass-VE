## 📱 Formato de Documentos

Todos los documentos están en **Markdown** (.md):
- Legibles en cualquier editor de texto
- Convertibles a PDF (Google Docs, Word, Markdown to PDF)
- Compartibles por email
- Versionables en Git

**Para convertir a PDF:**
```bash
# Opción 1: Google Docs
1. Copiar contenido
2. Pegar en Google Docs
3. Descargar como PDF

# Opción 2: Pandoc (si tienes instalado)
pandoc archivo.md -o archivo.pdf

# Opción 3: Online converter
Ir a: https://markdowntopdf.com
```
